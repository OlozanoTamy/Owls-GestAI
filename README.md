            <h2>Modelos Iniciales</h2>
            <p>Se han creado tres modelos iniciales para este proyecto:</p>
            <ul>
                <li><strong>Modelo Denso:</strong> Un modelo de red neuronal completamente conectada con capas densas.</li>
                <li><strong>Modelo Convolucional (CNN):</strong> Un modelo convolucional con varias capas de convolución y pooling.</li>
                <li><strong>Modelo Convolucional con Regulación (CNN2):</strong> Una versión mejorada del modelo convolucional que incluye capas de regularización como Dropout para prevenir el sobreajuste.</li>
            </ul>
        </div>
